## Scenario 1: Single File Edit

**Changes:** Modified `README.md` (+15/-3)

| Style | Message |
|-------|---------|
| **Conventional** | `docs: update readme.md`<br>`- 1 file changed: 15 insertions(+), 3 deletions(-)` |
| **Simple** | `Update README.md` |
| **Detailed** | `docs: Update 1 file`<br><br>`Modified: 1 md file`<br><br>`Changes: 15 additions, 3 deletions`<br><br>`Files:`<br>`  - README.md` |

## Scenario 2: New Feature (Multiple Components)

**Changes:**
- `src/components/Button.tsx` (+120)
- `src/components/Button.test.tsx` (+45)
- `src/styles/button.css` (+30)

| Style | Message |
|-------|---------|
| **Conventional** | `feat(components): update 3 files`<br>`- 3 files changed: 195 insertions(+), 0 deletions(-)` |
| **Simple** | `Update 3 files (+195/-0): Button.tsx, Button.test.tsx and 1 more file` |
| **Detailed** | `feat: Update 3 files`<br><br>`Modified: 1 tsx file, 1 test file, 1 css file`<br><br>`Changes: 195 additions, 0 deletions`<br><br>`Files:`<br>`  - src/components/Button.tsx`<br>`  - src/components/Button.test.tsx`<br>`  - src/styles/button.css` |
